# 方法说明

2015年和2016年分别取244个连续的交易日，244天的涨跌幅组成一个向量。 所以有2015年和2016年两个向量 ：Vec_2015 和 Vec_2016。

道琼斯的时间范围是1950年到2012年， 恒生指数的时间是2000年至今。这两种数据的取向量方法如下:

>假设所有数据组成了一个数列[p0, p1, p2, .... ,pn]

>第一个向量是[p1, ..., p224]

>第二个向量是[p2, ..., p225]

> .......


然后利用sklearn的算法包计算距离Vec_2015和Vec_2016最近的四个向量。
